# Swimming Performance Prediction (ML Project)

## Overview
This project predicts a swimmer's pace per 100m based on biometric data (age, height, weight) and race distance. It uses real-world data collected from the **Paris 2024 Olympics**.

## Key Features
* **Data Scraping:** Collected data using BeautifulSoup from official sources.
* **ML Models:** Compared Linear Regression, Random Forest, and XGBoost.
* **Pipeline:** Automated data preprocessing with Scikit-learn.
* **Interface:** Built an interactive UI using Gradio.

## Tech Stack
* Python (Pandas, NumPy, Scikit-learn, XGBoost, BeautifulSoup)
* Gradio (Deployment)
